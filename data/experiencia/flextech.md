# Flextech — Proyecto freelance
## Categoría: experiencia
## Período: Abril — Mayo 2025

---

## El proyecto

Landing page completa para Flextech, desarrollada de forma independiente
como proyecto freelance — sin equipo, sin supervisión técnica.

---

## Alcance completo

El proyecto cubrió todo el ciclo, desde el diseño hasta la entrega en producción:

- **Diseño en Figma** — diseño propio de la landing, sin partir de un template existente
- **Desarrollo front-end** — HTML, CSS, JavaScript y PHP
- **Configuración del servicio de correos** — integración y puesta en funcionamiento
- **Estimación de horas** — cálculo previo del tiempo a dedicar para definir el precio del proyecto

---

## Lo que implica trabajar solo en un proyecto completo

Manejar un freelance de punta a punta requiere más que código:
saber cuántas horas lleva cada etapa, traducir eso a un precio justo,
y entregar algo que el cliente pueda usar sin depender de nadie más para mantenerlo.

En este proyecto Gastón gestionó esa cadena completa de forma autónoma.

---

## Stack

- Figma — diseño UI
- HTML5, CSS3, JavaScript — front-end
- PHP — lógica del servidor y servicio de correos
